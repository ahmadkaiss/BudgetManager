## Budget Manager

### Data Structure:
The budget manager stores financial transactions in a dictionary:

```python
budget_data = {
   "metadata": {
        "balance": 500,
        "payment_method": "Credit Card"
   }

    "transactions": [
        {
         "type": "income",
         "cashflow": 1000,
         "label": "Salary"
       },
        {
           "type": "expense",
           "cashflow": 500,
           "label": "Groceries"
        }
    ]
}
